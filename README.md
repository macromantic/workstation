
# Workstation

This is my DevOps workstation environment installation and configuration.

## How to Install

``` bash
mkdir dev
cd dev
git clone https://github.com/macromantic/workstation.git
```

## How to Use

### Run the Bootstrap

``` bash
cd ~/dev/workstation
./bootstrap.sh
```

### Run the Google Cloud SDK

``` bash
cd ~/dev/workstation
./google-cloud.sh
```

Close the Terminal

### Run the Sublime License Configuration

```[bash]
cd ~/dev/workstation
./config-license.sh
```
